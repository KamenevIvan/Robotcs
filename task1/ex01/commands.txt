===
Название: cool-retro-term
Описание: Лучшая версия терминала
Синтаксис: cool-retro-term [--default-settings] [--workdir <dir>] [--program <prog>] [-p|--profile <prof>] [--fullscreen] [-h|--help]
Пример: cool-retro-term --fullscreen
Примечание: Всё работает как в классическом терминале, но при этом выглядит лучше

===
Название: cowsay
Описание: Выводит на экран животное с заданным сообщением
Синтаксис: cowsay [-bdgpstwy] [-h] [-e eyes] [-f cowfile] [-l] [-n] [-T tongue] [-W wrapcolumn] [message]
Пример: cowsay -f dragon Moooo
Примечание: -f позволяет заменить говорящий субъект. Также есть модификация cowthink - суть, я думаю, понятна.

===
Название: sl
Описание: Выводит на экран steam locomotive
Синтаксис: sl [-F] [-l]
Пример: sl -l
Примечание: -F - опция заставит поезд взлететь. -l - сделает поезд меньше

===
Название: toilet
Описание: Позволяет выводить на экран написи в формате больших текстовых баннеров.
Синтаксис: toilet [ -hkostvSW ] [ -d fontdirectory ] [ -f fontfile ] [ -F filter ] [ -w outputwidth ] [ -I infocode ] [ -E format ] [ message ]
Пример: toilet -f bigmono9 "Hello"
Примечание: bigmono9 - один из доступных форматов текста

===
Название: apt-get moo
Описание: Пасхалка
Синтаксис: apt-get moo
Пример: apt-get moo
Примечание: -

===
Название: rig
Описание: Генератор случайных (фейковых) персональных данных
Синтаксис: rig [-f | -m ] [ -d datadir ] [ -c num ]
Пример: rig 
Примечание: -f/-m позволяет выбрать пол

===
Название: w
Описание: Позволяет увидеть всех залогинившихся пользователей и некоторые дополнительные параметры
Синтаксис:  w [options]
Пример: w -h
Примечание: -h отключает выведение заголовков, информация становится более читаемой

===
Название: tail
Описание: Позволяет вывести n последних строк или байт файла.
Синтаксис:  tail [OPTION] [FILE]
Пример: tail -f file.txt
Примечание: опция -f позволяет в реальном времени выводить новые данные в файле.

===
Название: ps 
Описание: Выводит список текущих процессов
Синтаксис: ps [options]
Пример: ps -A
Примечание: -A выводит полный список всех процессов

===
Название: du
Описание: Позволяет посмотреть, сколько места на диске занимает дерево директорий
Синтаксис: du [OPTION] [FILE]
Пример: du -h Documents/
Примечание: -h - представить размер в человеко-читаемом формате (1K, 234M, 2G)
